Stage 0: Topic Proposal

Topic Overview: My idea is to create a web app that allows students or residents to buy, trade, or sell items locally. Also local business can offer thier goods and services on this web app. I got inspired by facebook market, but I wants to make a easier version for students to trade books, I think it also will help local business to grow.

What else this app does: This web app also allows user to create "looking for an item". If the user is trying to look for something that is currently not on the web, they can create a notification, if there is someone else posting the item for sell, the app will notify the user.

Advantage: This app allows local buyer and seller to do business. And It also allows seller to create bid, so whoever bid the highest is the winner. Same day pick up without any shipping fees.
